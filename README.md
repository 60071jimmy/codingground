# codingground
Main Repository for Coding Ground

codingground程式測試區

codingground網址：

https://www.tutorialspoint.com/codingground.htm

C語言程式專案編譯預設指令：

gcc -o main *.c

可以藉由64位元的gcc編譯：

x86_64-redhat-linux-gcc -o main *.c

## 使用library
### 使用math.h library
> gcc -o main *.c -lm
> x86_64-redhat-linux-gcc -o main *.c -lm
